## Overview

This repository documents the full research, modelling, and validation work undertaken for **EGH490-2 Research Project 2**, completed as part of the Bachelor of Engineering (Honours).

The project builds upon prior work from EGH490-1, extending both analytical depth and simulation fidelity to develop a technically rigorous and validated system design.

The work represents a complete engineering research lifecycle:

- Problem formulation  
- Literature synthesis  
- Analytical modelling  
- Numerical simulation  
- Design iteration  
- Validation & verification  
- Final technical reporting  

---

## Research Focus

This project investigates the **design and analysis of a waveguide slot array antenna**, with emphasis on:

- Electromagnetic field behaviour in rectangular waveguides  
- Slot excitation and coupling mechanisms  
- Parametric influence of slot offset and geometry  
- Array synthesis and radiation control  
- Gain, directivity, and efficiency optimisation  
- Structural feasibility and manufacturability

## Technical Scope

The repository covers core electromagnetic and antenna engineering concepts, including:

- Waveguide mode analysis (TE modes)  
- Slot admittance and coupling theory  
- Array factor formulation and synthesis  
- Radiation pattern modelling  
- Impedance behaviour and matching considerations  
- Numerical validation against analytical predictions  

All assumptions, boundary conditions, and modelling constraints are explicitly documented.

---

## Methodology

The project follows a structured engineering workflow:

1. Derivation of waveguide field equations  
2. Extraction of slot self-admittance via simulation  
3. Parametric sweeps (e.g. slot offset, cavity length)  
4. Integration into array synthesis framework  
5. Radiation pattern optimisation  
6. Validation against theoretical expectations
